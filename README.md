# WebScraping-and-Visualization-Nutrition-Food-Analysis
Web Scraping ข้อมูลอาหารและโภชนาการจากเว็บไซต์ด้วย Python (BeautifulSoup)
Data Cleaning และจัดหมวดหมู่อาหารและวัตถุดิบตามประเภทโภชนาการ
วิเคราะห์ข้อมูลและสร้าง Visualization ด้วย powerbi เพื่อเปรียบเทียบอาหารที่ส่งผลดีและไม่ดีต่อสุขภาพ
สรุป Insight เพื่ออธิบายรูปแบบพฤติกรรมการบริโภคอาหาร
